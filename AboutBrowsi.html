Browsi - The lightweight Python browser!

Welcome to Browsi - The simple, lightweight, Python-based web browser!



Cool things about browsi:

* Browsi is EXTREMELY lightweight (under 200 lines of code)
* It runs on ONLY the packages provided in the DEFAULT python distribution!
* really fast
* It's super simple!



Less cool things about browsi:

* horrible HTML 5 support
* horrible CSS support
* No JS
* No Plugins
* EXTREMELY bare bones
* can't quite follow links yet(working on it!)
